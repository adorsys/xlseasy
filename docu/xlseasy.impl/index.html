Implementation

xlseasy.impl uses the components xlseasy.api and xlseasy.cbe to run xlseasy.

Packages

  • converter
  • proc
converter takes care to make the necessary conversions for :
  • cell's data (BigDecimalCellConverter, BooleanCellConverter, ...).
  • collection's element and type (CollectionElementConverter, CollectionTypeConverter, GenericCollectionElementConverter, ...)
  • and sheets (SheetConverter, SheetConverterKey)
proc handles the spreadsheet element's descriptor, session and style such as:
  • ColumnDesc implements ColumnDescIF (Descriptor): used to manages column properties (annotation, style, property's name, ...).
  • SheetDesc implements SheetDescIF (Descriptor): creates, gets and formats the sheet.
  • WorkbookStyle: used to set and get the workbook's style.
